=== Descipción: ==========================
	- Cada peticion de le genera un UUID para la peticion, cada una registra y se procesa individualmenta cada uno de los movimientos o ventas contenidas en ella.
	- Cada movimiento se envia a SAP y se registra la bitacora
	- Cada venta se guarda en EPSSTG.POSDM_VENTAS_SOFTREST y se registra la bitacora

=== Despliegue: ==========================
Dulceria generar Jar (Recomendacion: eliminar la carpeta target antes de generar version)
* Produccion  
	- mvn clean package -P prod
	- nohup java  -Xms1024M -Xmx2048M -jar dulceria-api.jar &
* QA
	- mvn clean package -P qa -Dskiptests=true
	- nohup java -jar dulceria-api-qa.jar &
